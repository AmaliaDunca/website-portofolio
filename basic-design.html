About me

Theme description

The Task given:
Build a responsive website based on Pixel Art style.

During this theme I’ve covered basic web design from research and idea generation to design and communication theory.I also get my hands on html and css coding as well as basic image processing (Photoshop) and vector drawing (Illustrator).

Achieved goals for each module in the theme:

Module 1: Content and Communication

First step was to learn about digital media and communication, to do research about Pixel art style and to create some content. To Get a clear vision of what is the feeling of Pixel art style I made a moodboard.

Communication and Presentation

Next step was collecting, optimising, and producing images and text.

Design and visualisation

Get to know the basic principles for the composition and layout of digital design in multimedia production helped me to make the first sketches and process from lots of drafts to final product

Module 2: Basic HTML and CSS

Final step was to structure content using HTML, presenting content using CSS and some more advanced HTML5 and CSS3, including media queries, responsive navigation, advanced selectors...for creating a responsive web design.
